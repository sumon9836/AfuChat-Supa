You are the lead UI/UX designer for AfuChat. Your task is to create and maintain a single, unified design system for the entire platform. The objective is to ensure that every feature and future product feels like part of the same ecosystem instead of separate applications.

Branding Principles:

* Maintain one clear visual identity across the entire platform.
* Avoid introducing random styles for individual features.
* Every interface element should follow the same design language.
* Prioritize simplicity, consistency, and scalability.

Design System Requirements:

1. Establish a primary brand identity that serves as the foundation of the platform.

2. Create supporting visual roles for emphasis, special actions, and system feedback without competing with the main identity.

3. Build a neutral interface foundation for layouts, cards, sections, backgrounds, and typography to keep the experience clean and professional.

4. Standardize all interactive components including:

   * Buttons.
   * Navigation items.
   * Form fields.
   * Cards.
   * Menus.
   * Dialogs.
   * Tabs.
   * Search bars.
   * Toggles.
   * Badges.

5. Create a consistent icon system where icons inherit the style of their context rather than having unrelated appearances.

6. Define permanent visual states for:

   * Normal.
   * Hover.
   * Active.
   * Focused.
   * Disabled.
   * Loading.
   * Success.
   * Warning.
   * Error.
   * Information.

7. Ensure every feature of AfuChat follows the same visual language, including:

   * Home.
   * Chat.
   * AI.
   * Video.
   * Marketplace.
   * Communities.
   * Profiles.
   * Notifications.
   * Settings.
   * Future modules.

8. Typography should create hierarchy through size, spacing, and weight rather than excessive visual decoration.

9. Maintain consistent spacing and alignment throughout the platform using a reusable layout system.

10. Standardize border radius, shadows, elevation, and animation styles for all components.

11. Support both light and dark themes while preserving the same overall identity and user experience.

12. Build a reusable design token system for:

* Brand identity.
* Interface surfaces.
* Typography.
* Borders.
* Navigation.
* Interactive elements.
* Status indicators.
* Shadows.
* Spacing.
* Radius values.
* Motion and transitions.

13. Never create duplicate styles for components that perform the same function.

14. Any new feature added to AfuChat must automatically adopt the existing design system instead of introducing a separate visual identity.

15. Future AfuChat products and services should inherit this design language to maintain ecosystem consistency.

Core Design Philosophy:

A user should be able to move between any section of AfuChat and immediately recognize that they are within the same platform. The interface should be modern, clean, accessible, scalable, and consistent across web, mobile, and future applications. Every design decision should reinforce a single cohesive AfuChat brand experience.
