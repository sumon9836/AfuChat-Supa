🚀 AfuChat Animation & UI Motion Design Prompt (For Developers)

Build a consistent animation system for the AfuChat platform (mobile app + web) using a minimal Corporate Memphis-inspired illustration style that matches the existing AfuChat brand identity.

🎨 Visual Style Requirements
Use flat vector illustrations (Corporate Memphis style)
Characters should be:
Simple, friendly, rounded shapes
No realistic textures or gradients on characters
Vibrant solid colors only (brand-aligned palette)
Maintain clean tech UI aesthetic
Backgrounds must remain white or very light neutral tones
All illustrations must feel like part of the same design system
🧠 Brand Consistency Rules (AfuChat Identity)
Every animation must reflect AfuChat as:
A modern communication/social platform
Friendly, global, lightweight tech brand
Use AfuChat brand colors consistently (primary, secondary, accent)
UI elements (chat bubbles, icons, loaders) must match AfuChat design system
Avoid random art styles or mixed illustration types
🎬 Animation Guidelines

Implement smooth micro-interactions and lightweight animations:

Page transitions (fade, slide, scale)
Chat message animations:
Incoming: slide + fade in
Outgoing: pop + slight bounce
Loading states:
Animated dots / wave loaders in AfuChat colors
Empty states:
Friendly toon illustration (like user working, chatting, connecting)
Button interactions:
Soft scale-down on press
Subtle glow or ripple effect
🧩 Illustration Usage

Use toon-style characters like:

Users chatting on laptop/phone
Developers working on AfuChat
Global connection/network scenes
Social interaction visuals

All illustrations must be:

Consistent perspective
Same stroke/shape style
Same character proportions across the app
⚙️ Technical Implementation Suggestions
Use Lottie animations for lightweight vector motion
Use Framer Motion (React Native / Web) or equivalent animation library
Optimize for low-end Android devices (important for Uganda + global users)
Keep animations under 200–400ms where possible for responsiveness
📱 UX Goals
Make AfuChat feel:
Fast ⚡
Friendly 😊
Modern 🌐
Lightweight 📱
Animations should enhance UX, not distract
🔥 Important Rule

No animation should break usability or slow down navigation. Performance comes first.
